## Overview

The AI Insights panel transforms raw sentiment data into actionable strategic recommendations. It analyzes emotion distributions, crisis levels, and volume trends to generate professional reports tailored for PR teams, communications managers, and decision-makers.

## How It Works

<Steps>
  <Step title="Select a Topic">
    Open any topic from the dashboard. The AI Insights panel appears on the right side of the Topic Detail view.
  </Step>
  
  <Step title="Generate Report">
    Click **"Generate Report"** to trigger AI analysis. The system automatically:
    - Pulls current sentiment score
    - Analyzes emotion breakdown
    - Identifies crisis level and volume trends
    - Streams a strategic analysis in real-time
  </Step>
  
  <Step title="Review Recommendations">
    The generated report includes:
    - **Situation Assessment**: Current state summary
    - **Key Concerns Identified**: Main issues driving sentiment
    - **Recommended Actions**: 4-5 specific, implementable steps
    - **Opportunities**: Positive angles to leverage
    - **Risk Factors to Monitor**: Escalation triggers
  </Step>
  
  <Step title="Ask Follow-Up Questions">
    Use the interactive chat to dive deeper:
    - "What should we prioritize first?"
    - "How can we address the anger?"
    - "What are the quick wins?"
  </Step>
</Steps>

## Report Structure

Every AI-generated report follows this professional format:

### 1. Situation Assessment

```markdown
## Situation Assessment
Public sentiment around **AI Regulation Debate** is currently **divided**, 
with **fear (35%)** as the dominant emotion. Crisis level is **medium**, 
indicating elevated public concern. Volume has shifted **+42%**, suggesting 
rapidly escalating public engagement.
```

**Purpose**: Provides a 2-3 sentence snapshot of the current landscape.

### 2. Key Concerns Identified

```markdown
## Key Concerns Identified
- **Fear dominance (35%)** — the primary emotional driver shaping narratives
- **Anger** as secondary emotion signals competing public reactions
- Rising volume indicates growing public interest that needs to be channeled
- Emotional polarization (fear 35%, anger 28%, surprise 18%) suggests fragmented audience segments
```

**Purpose**: Bullet-point list of the 3-4 main issues driving sentiment.

### 3. Recommended Actions

```markdown
## Recommended Actions
1. **Monitor and document** — track the fear signals in real-time for early escalation warnings
2. **Engage proactively** — address the leading concerns with transparent, factual communication via official channels
3. **Segment your response** — tailor messaging to the distinct emotional groups (fear vs anger audiences)
4. **Establish rapid response protocols** — Proactive monitoring and engagement is advised.
5. **Measure sentiment shift** — set up hourly tracking alerts for #AIRegulation
```

**Purpose**: 4-5 specific, actionable steps. Each starts with an action verb and is implementable.

### 4. Opportunities

```markdown
## Opportunities
- The **35% fear** signal can be redirected into constructive dialogue if addressed proactively
- Current volume spike (+42%) is an opportunity to own the narrative before it solidifies
- Building credibility now through transparent communication will pay dividends when sentiment stabilizes
```

**Purpose**: 2-3 positive angles or silver linings.

### 5. Risk Factors to Monitor

```markdown
## Risk Factors to Monitor
- **Escalation trigger**: A new development could push sentiment toward crisis level
- **Narrative hijacking**: Third parties or media may amplify the anger signal to shift the story
- **Sentiment fatigue**: Prolonged high-volume discourse without resolution leads to disengagement or backlash
```

**Purpose**: 2-3 things that could make the situation worse.

    ```typescript
      const userMsg: Message = {

      };

      streamInsights({

        isChat: true,  // Flag for conversational mode
        onDelta: (chunk) => {
          assistantContent += chunk;
          setChatMessages((prev) => {
            const last = prev[prev.length - 1];
            if (last?.role === 'assistant') {
              return prev.map((m, i) => 
                i === prev.length - 1 
                  ? { ...m, content: assistantContent } 
                  : m
              );
            }
            return [...prev, { 
              role: 'assistant', 
              content: assistantContent 
            }];
          });
        }
      });
    };
    ```

## Configuration

To enable AI insights, add at least one API key to your `.env` file:

```bash
# Recommended: OpenAI (best quality)
VITE_OPENAI_API_KEY=sk-proj-...

# Alternative: Gemini (free tier available)
VITE_GEMINI_API_KEY=AIza...

# Fallback: Groq (fast inference)
VITE_GROQ_API_KEY=gsk_...
```

<Info>
**No API keys?** The platform still works! Local generation is always available and provides solid template-based analysis.
</Info>

## Use Cases

### Crisis Response Planning

**Scenario**: Food price crisis (48% anger, 22% fear)

**AI Recommendation**:
```markdown
## Recommended Actions
1. **Acknowledge public frustration** — Issue statement recognizing grocery cost burden
2. **Share concrete relief measures** — Announce price control investigations, subsidies, or relief programs
3. **Combat misinformation** — Address "corporate greed" narrative with supply chain transparency
4. **Engage directly with affected communities** — Town halls, social media Q&A sessions
```

```markdown
## Opportunities
```

## Limitations

<Note>
**LLM hallucination risk**: AI-generated recommendations are based on emotion percentages and keywords, not deep semantic understanding. Always cross-check with the raw Live Feed and Top Voices panels.

**No domain expertise**: The AI doesn't know your industry, brand history, or internal constraints. Treat recommendations as a starting point, not gospel.

**Static analysis**: Reports are generated from a single snapshot. For evolving situations, regenerate every 6-12 hours.
</Note>
